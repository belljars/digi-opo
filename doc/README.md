# digi-opo dokumentaatio

Tämä kansio sisältää projektin perusdokumentaation suomeksi.

Dokumentit:

- `kaynnistys.md`: asennus, käynnistys, Nix-käyttö ja yleisimmät kehityskomennot
- `arkkitehtuuri.md`: projektin rakenne, pääkomponentit, datan kulku ja pysyvä tallennus
- `asetukset-ja-piilotus.md`: asetussivu, globaali piilotus ja sen vaikutus koko sovellukseen

Projektin varsinainen lähdekoodi sijaitsee pääasiassa kansioissa `src/`, `tests/`, `data/` ja `user/`.
