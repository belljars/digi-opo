# digi-opo dokumentaatio

Tämä kansio sisältää projektin perusdokumentaation suomeksi.

Dokumentit:

- `kaynnistys.md`: miten sovellus asennetaan ja käynnistetään paikallisesti
- `arkkitehtuuri.md`: projektin rakenne, pääkomponentit ja datavirta
- `asetukset-ja-piilotus.md`: asetussivu, globaali piilotus ja sen vaikutus koko sovellukseen

Projektin varsinainen lähdekoodi sijaitsee pääasiallisesti kansioissa `src/`, `data/`, `tests/` ja `user/`.
